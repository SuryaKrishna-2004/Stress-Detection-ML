# Stress Detection Web Application

This project is a web application that detects stress levels using machine learning and facial image analysis. Developed as part of my BCA degree project.

---

## Technologies Used
- Python
- Flask (Web Framework)
- HTML, CSS
- OpenCV (Image Processing)
- Machine Learning
- SQLite Database

---

## Project Overview
The system analyzes facial images and user input to identify stress patterns. It is designed to support workplace wellness monitoring and productivity awareness.

---

## Key Features
- Web-based user interface
- Stress detection using image processing
- Machine learning model integration
- Data storage using database
- Simple and user-friendly workflow
